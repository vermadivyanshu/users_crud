## Team Members CRUD

### Installation
#### Pre requisties
Installing pipenv using pip
`pip install pipenv`
From the root directory, run the following:
`pipenv install`

### Running tests
`python manage.py test`

### Running development server
`python manage.py migrate`
`python manage.py runserver`
